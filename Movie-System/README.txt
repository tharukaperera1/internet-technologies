CINESCOUT - MOVIE DISCOVERY WEB APPLICATION
============================================

Project for API-Based Web Application (Final Project)

API Used: OMDb API (Open Movie Database)
API Key: 1c768e4f (free tier - works for educational purposes)

FEATURES IMPLEMENTED:
1. Movie Search Functionality - Search by exact movie title
2. Random Movie Generator - "Discover a Movie" button
3. Browse by Keyword - Search using general keywords
4. Browse by IMDb ID - Fetch specific movie by ID
5. Movie Details Page - Modal with full movie details
6. Top Rated / Popular Movies Section - Curated list

ADDITIONAL FEATURES:
- Responsive design (works on mobile, tablet, desktop)
- Modern dark theme UI
- Smooth animations and hover effects
- Keyboard Enter support for all search inputs
- Loading states and error handling

HOW TO USE:
1. Open index.html in a web browser (or deploy to web server)
2. Use search cards to find movies by title, keyword, or IMDb ID
3. Click "Random Movie" for surprise recommendations
4. Click "Top Rated / Popular" to refresh curated list
5. Click any movie card to see full details in modal popup

TECHNOLOGIES:
- HTML5
- CSS3 (Flexbox, Grid, animations)
- Vanilla JavaScript (ES6+)
- OMDb REST API

DEPLOYMENT:
To deploy on GitHub Pages:
1. Create a new repository on GitHub
2. Upload all 4 files (index.html, style.css, script.js, README.txt)
3. Go to Settings > Pages > Select branch (main) > Save
4. Access your site at: https://[username].github.io/[repo-name]
